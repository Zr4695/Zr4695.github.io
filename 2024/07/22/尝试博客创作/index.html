#9、嵌入
嵌入图片
法1（没有空格）
！[ 图片名称 ] ( 图片地址，可以是网站的地址 )
法2，直接拖进来

嵌入链接
法1
直接复制粘贴
法2（与图片相比少了！）
[ 图片名称 ] ( 网站的地址 )

#10、代码
两个反引号
code
use `code` in you markdown file.
代码块，用三个反引号

#11、脚注
[ ^脚注代号] 用[ ^ 1]标识 （去掉空格）
即为这个样子 [^1]
[^脚注代号]+:+空格+脚注内容
[^1]: 脚注内容 多段落用转行加tab

#12、分割线
三个或多个（ *** ）（ — ）（___ ） 三个破折号需要与上一行空一行
***

#12、图片大小修改

![[风景1.jpg]]
![[风景1.jpg|200]]
![[风景1.jpg|400x200]]
